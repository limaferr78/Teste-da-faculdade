Primeiro projeto feito em HTML

Breve explicação sobre o projeto

Esse é um teste para um trabalho de faculdade, onde o objetivo é criar um site simples usando HTML.
Estou aprendendo a estruturar uma página web e a usar as tags básicas do HTML para criar conteúdo.
Também utilizei a ajuda do COPILOT que já vem nativo no VS Code, me ajudando a corrigir erros e implementar novas funcionalidades.

Exemplos de formatação de texto:

Para deixar algo em negrito fazemos assim: negrito (a fórmula usada foi <b></b>)

Ou assim: Fica dessa forma (a fórmula usada foi <b<strong><strong></b>)

Para deixar algo em itálico fazemos assim: itálico (a fórmula usada foi <i></i>)

Ou assim: Fica dessa forma (a fórmula usada foi <i><em><em></i>)

Para sublinar algo, fazemos assim: Fica assim (a fórmula usada foi <u></u>)

Agora por exemplo, podemos sublinar e deixar em negrio ao mesmo tempo: Fica dessa forma

Para destacar algo, fazemos assim: Fica assim (a fórmula usada foi <mark></mark>)

Agora por exemplo, podemos destacar e deixar em negrio ao mesmo tempo: Fica dessa forma
