Primeiro projeto feito em HTML

Breve explicação sobre o projeto

Esse é um teste para um trabalho de faculdade, onde o objetivo é criar um site simples usando HTML. Estou aprendendo a estruturar uma página web e a usar as tags básicas do HTML para criar conteúdo. Também utilizei a ajuda do COPILOT que já vem nativo no VS Code, me ajudando a corrigir erros e implementar novas funcionalidades.

Exemplos de formatação de texto

Negrito: negrito (<b>) ou strong (<strong>).

Itálico: itálico (<i>) ou ênfase (<em>).

Sublinhado: sublinhado (<u>). Combinando: negrito e sublinhado.

Destacar: marcado (<mark>). Combinando: negrito e marcado.
